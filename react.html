APRENDIENDO WEBPACK

Webpack sin configuración

instalar las dependencias en desarrollo yarn add webpack webpack-cli --dev
Configurar el script "build": "webpack"-d
Es importante tener en cuenta el modo a desarrollar ya sea en modo producción o modo de desarrollo
Importante: Conocer los puntos de entradas (--entry .direccion.) y salidas (--output-path .direccion.).

Babel :o

1. Instalar las dependencias en dev necesarias para trabajar con babel yarn add babel-loader @babel/core @babel/preset-env --dev
2. Crear un archivo .babelrc y hacer sus respectivas configruaciones
3. Crear un archivo webpack.config.js y hacer sus respectivas configuraciones

Inyección de Js en HTML

1. Instalar las dependencias de desarrollo yarn add html-loader html-webpack-plugin --dev

Estracción de css

1. Instalar las dependencias de desarrollo yarn add mini-css-extract-plugin css-loader style-loader --dev

Servidor web de desarrollo

1. Instalar las dependencias de desarrollo yarn add webpack-dev-server --dev

Manejo de archivos

1.1 Configurar el type = 'assets' para poder trabajar con files ó 1.2 Instalar las dependencias de desarrollo yarn add file-loader --dev

Optimización de imagenes

1. Instalar las dependencias de desarrollo yarn add image-webpack-loader --dev

Multiples Puntos de entradas

Se debe tener en cuenta el punto de entrada y punto de salida y tambien el HtmlWebpackPlugin

Hello World con Type Script (ts)

1. Instalar las dependencias de desarrollo yarn add typescript ts-loader --dev
